# Painel Cozinha Saí

Sistema de gestão de produção e estoque para cozinha industrial.

## Funcionalidades

### Painel Gestor
- Dashboard com KPIs
- Gestão de Estoque (Insumos, Preparos, Sensível)
- Fichas Técnicas com controle de visibilidade
- Gestão de Tarefas e Equipe
- Quadro de Produção com calendário

### Painel Operacional
- Checklist de Deveres diários
- Kanban de Produção
- Contagem de Estoque Sensível
- Fichas Técnicas de Preparo

## Tecnologias
- React 19 + TypeScript
- tRPC + Express
- Drizzle ORM + MySQL/TiDB
- Tailwind CSS 4
